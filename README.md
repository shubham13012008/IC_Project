# IC_Project
I am making my IC project with my team code on duty
We have made a Travel Management System Project in C.

Our project aims at the problem of train ticket booking and management.
We have implemented our knowledge of the C language to make the project which we can use for the booking of the trains.
We have used multiple concepts that we have studied in the class to make this project like sorting, structures,loops and many more.
It has multiple features like searching for trains, booking tickets, cancelling tickets, adding trains, deleting trains, editing train details, viewing stats, etc.
We have used alogrithm to search for the trains in case if a direct train is not available then the code searches for the connecting train that can get the passenger to their destination and we also sort the connecting trains if there are more than one then we show the trains according to the ascending order of prices.


It has 2 Portals :-
1. User Portal
2. Admin Portal

User Portal has 3 features :-
1. Search Trains
2. Book a Ticket
3. Cancel a Ticket

Admin Portal has 4 features :-
1. Add a Train
2. Delete a Train
3. Edit Train Details
4. View Stats
